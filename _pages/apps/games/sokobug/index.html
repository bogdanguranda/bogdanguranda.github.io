Summary

Sokobug is a simple 2D puzzle game inspired by Sokoban (1988). It was implemented using libGDX and Java by Bogdan and Andrei Guranda as part of the Potatoes Games project.

Gameplay

Push boxes onto target tiles to clear levels. The game features handcrafted levels and a minimal, retro graphical style.

Controls

  • Arrow keys / WASD — move
  • R — restart level
  • Esc — back to menu

Technologies

  • Language: Java
  • Engine / Framework: libGDX
  • Export targets: Desktop (Windows/macOS/Linux), HTML build available
