Summary

Trading Assistant is a comprehensive trading analysis and portfolio management platform. It combines market research, technical analysis, and portfolio tracking to help traders and investors make informed decisions.

Features

  • Portfolio Tracking — Monitor holdings with real-time P&L and performance metrics
  • Technical Analysis — Advanced charting with multiple indicators and pattern recognition
  • Trade Journal — Log and analyze your trades for continuous improvement
  • Risk Management — Position sizing, stop-loss recommendations, and exposure analysis
  • Market Research — Integrated news, earnings data, and fundamental analysis tools
  • Alerts & Notifications — Customizable alerts for price levels, technical setups, and market events

Technologies

  • Language: Python / TypeScript
  • Data APIs: Yahoo Finance, Alpaca, IB API
  • Database: PostgreSQL
  • Frontend: React / Vue
