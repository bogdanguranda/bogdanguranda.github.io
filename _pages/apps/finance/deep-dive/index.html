Summary

Deep Dive is a powerful desktop application for stock market research and analysis. Built with modern web technologies and packaged as a native Electron app, it provides traders and investors with tools to search stocks, analyze price movements with interactive charts, create custom watchlists, and organize market research through custom formulas and scans.

Features

  • Ticker Search & Details — Quickly search and view detailed information about US stocks and ETFs
  • Interactive Charts — Advanced charting with support for multiple technical indicators, moving averages, and customizable analysis tools
  • Smart Watchlists — Create and manage custom watchlists; includes built-in watchlists for all US stocks, ETFs, and non-ETF stocks
  • Custom Formulas — Define your own formulas for complex analysis and calculations
  • Market Scans — Create scans to filter stocks based on custom criteria
  • Live Data Sync — Automatic data synchronization from Yahoo Finance and Alpaca API
  • Market Status — Track real-time market session information
  • Historical Data — SQLite-backed database for storing and querying historical daily price data

Technologies

  • Language: TypeScript
  • Desktop Framework: Electron 39
  • UI Framework: React 19
  • Charting Library: ECharts 6
  • Build Tool: Vite
  • Database: SQLite
  • Math Engine: math.js
  • Package Manager: pnpm
  • Export Targets: Desktop (Windows/macOS/Linux)
