Summary

Trading Assistant Discord Bot is a Python-powered automation tool for traders and investors who want market data, stock quotes, RSS-driven news, and Alpaca order support directly in Discord. It integrates market calendars, live alerts, and research workflows so your community can stay informed with minimal setup.

Features

  • Discord Alerts — Send market signals, watchlist updates, and research prompts to configured Discord channels
  • Ticker Quotes — Fetch live quotes and company data for US equities through supported market APIs
  • RSS News Feed — Parse and relay trading-related news from configurable RSS sources
  • Alpaca Support — Optional Alpaca API integration for account, data, and trading workflow support
  • Market Calendar — Use exchange calendars to compute trading session times and market status
  • Python Automation — Built with modern Python tooling for easy local development and deployment

Technologies

  • Language: Python 3.12+
  • Bot Framework: discord.py / discord
  • Data APIs: Alpaca, Finnhub (optional), RSS feeds
  • Scheduling: background jobs for market polling and RSS refresh
  • Deployment: PyInstaller live.spec for Windows executable packaging
  • Package Manager: Poetry
