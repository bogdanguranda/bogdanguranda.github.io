Summary

This is an app that generates aphorism images from a collection of curated sayings stored in files, adapts the font size dynamically based on the length of the aphorism to ensure optimal readability and aesthetic appeal. And then schedules posting to x.com using X API at predefined intervals.

Use cases

  • Single image generation of an aphorism image.

  • Single image generation of an aphorism image and posting to x.com using X API.

  • Multi-image generation of aphorism images from a file based collection.

  • Automated multi-image generation and scheduled posting to x.com using X API from a file based collection.

Technologies

Language

  • Python

Libraries

  • pandas: used for the files data manipulation.

  • pillow: used for image creation and manipulation.

  • tweepy: used for posting to x.com through X API.

  • schedule: used for scheduling post publishing to X.
